# 📏 CamRuler: Digital Calibration Tool

## 📖 Overview
A specialized utility application that utilizes camera pixel-to-metric ratios to calculate real-world physical distances. By calibrating the camera lens to a specific depth plane, this tool turns a standard webcam into an interactive digital ruler.

## 🛠️ Focus Areas
* **Mathematical Translation:** Converting raw pixel data into accurate metric outputs.
* **Interactive UX:** Allowing users to seamlessly select measurement points on a live feed.
* **Calibration Testing:** Iteratively refining the calculation formula for real-world accuracy.
